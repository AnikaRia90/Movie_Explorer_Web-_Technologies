Movie Info Explorer

Movie Info Explorer is a role-based web application developed for the Web Technology course using PHP, JavaScript, and CSS. The system runs on a local server environment using XAMPP (Apache & MySQL) and allows users to explore detailed movie information through an interactive interface with authentication and role management.

Project Overview

The platform is designed to present organized movie information where users can search, view details, manage favourites, give ratings, and share feedback. It follows a structured role-based workflow with Admin approval and user interaction features.

User Roles and Features
Admin

Full control over the movie database (add, update, delete)

Manage user accounts

Approve editor movie submissions

Moderate and delete user comments or reviews

Content Editor

Update movie information (cast, genre, details)

Submit new movies for admin approval

Tag movies by category or genre

Receive notifications after approval

Registered User

Search and view movie details

Add or remove favourites (watchlist)

Provide ratings

Write comments or reviews

Common System Features

User authentication (login, logout, registration)

Account management and profile settings

Personalized dashboard

Role-based access control

Technology Stack

Backend: PHP

Frontend: HTML, CSS, JavaScript

Server Environment: XAMPP (Apache)

Database: MySQL

Project Goal

This project focuses on building a structured movie information platform that demonstrates role management, approval workflows, and interactive web application development using core web technologies.
